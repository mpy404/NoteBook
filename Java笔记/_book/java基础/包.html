包的创建

  • 包其实就是文件夹
    • 作用
      • 对类进行分类管理
    • 定义格式
      • package 包名;（多级包用"."分开）
      • package com.it.ytr;
  • 包的创建
    • 手动创建包
      • 创建文件夹
    • 自动创包
      • javac -d .HelloWorld.java

包的导入

  • import 包的路径.类名
    • import java.util.Scanner
